# api.cameloz

api.cameloz is a project created as a documentation for the Backend API with Laravel.

## Requirements for development environment

- PHP >= 7.4
- Composer
- SQLite 3

## Install on local

```
cd path/to/somewhere
git clone https://github.com/jamband/api.cameloz.git
cd api.cameloz
composer run dev
```
